# Space Defense Game Prototype

This repository contains a small prototype of a 2D space defense game. The game
simulates a spaceship orbiting a planet and its moon using simple Newtonian
gravity.

## Requirements

- Python 3
- [Pygame](https://www.pygame.org/) library

## Running the Game

Install the `pygame` package and run `src/orbit_game.py`:

```bash
pip install pygame
python src/orbit_game.py
```

Use the arrow keys to apply thrust to the spaceship. A green line shows the
predicted orbit based on the current velocity.
